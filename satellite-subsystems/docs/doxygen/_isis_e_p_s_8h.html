int IsisEPS_saveConfig ( unsigned char  index,
unsigned short  checksum,
ieps_statcmd_t *  p_rsp_code 
)

Save iEPS configuration to non-volatile memory.

Parameters
[in]indexindex of iEPS I2C bus address.
[in]checksumCRC16 checksum over the entire read/write parameter list contained in memory using the CRC-CCITT-BR polynomial (f(x)=x**16 + x**12 + x**5 + 1) using 0xFFFF as seed if the checksum does not match the save command is rejected and the configuration is not saved. In that case a load operation should be performed to recall the known configuration before re-trying a save. Provide the value 0 to force a save, the checksum check will not be performed in this case.
[out]p_rsp_coderepresents the default response for every command.
Returns
