Detailed Description

The struct contains the possible first internal status variables for the TXS.

Field Documentation

struct { ... } _txs_status_data1_t::fields

Individual status bits of the TXS Status Byte.

unsigned char _txs_status_data1_t::PA_ENABLE

PA_ENABLE = '0' indicates that the power amplifier is inactive while PA_ENABLE = '1' indicates that the power amplifier is active.

unsigned char _txs_status_data1_t::PLL_LOCK

PLL_LOCK = 0x00 = PLL Inactive, = 0x01 = PLL Success, = 0x02 = PLL Failed, = 0x03 = PLL Busy.

unsigned char _txs_status_data1_t::rawValue

rawValue is the real value inside the union of the TXS Status Byte.

unsigned char _txs_status_data1_t::TX_ENABLE

TX_ENABLE = '0' indicates that the transmit functionality is being disabled. TX_ENABLE = '1' indicates that the transmit functionality is enabled.

unsigned char _txs_status_data1_t::TXS_MODE

TXS_MODE = 0x00 = Nominal, 0x01 = External.

unsigned char _txs_status_data1_t::TXS_STATE

TXS_STATE = 0x00 = OFF State, = 0x01 = ON State.
